## Ling Xi（灵希）管理系统简介
Ling Xi管理系统是一个基于Vue3 + Java开发的通用的后台管理系统，可复用此项目代码快速搭建一个后台管理系统。
同时会实现一个个人网站管理后台，包括：文章管理，微博管理，评论管理，订阅管理等基础功能。

该项目主要会维护两个分支版本。
- main: 最新分支，打通后端接口，功能最齐全，主要会实现一个个人网站的后台管理系统
- thin: 最小功能集合，其它项目可基于此分支快速搭建一个管理端的壳子
